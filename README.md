# ICT

This repository contains:

- `ictvisitorsfrontendsystem` — Next.js frontend
- `ICTVisitorsManagementSystem` — .NET backend

Deployment

See `DEPLOYMENT.md` for steps to deploy the frontend to Vercel and the backend to Azure App Service.

Local development

- Frontend: `cd ictvisitorsfrontendsystem` then `npm install` and `npm run dev`.
- Backend: open the `.sln` in Visual Studio or use `dotnet run` in the project folder.
